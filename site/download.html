Release assets are published through GitHub Releases. If there is not a packaged release yet, build from source with Deno 2.9 or newer.

Windows builds are expected as zipped `dist-desktop/DenoDesktopTldraw` bundles. macOS and Linux builds can be added as release automation grows.

Build from source

git clone https://github.com/EthanThatOneKid/deno-desktop-tldraw.git
cd deno-desktop-tldraw
deno task desktop:build
